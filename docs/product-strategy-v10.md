## 三、B 端定价（年付，必须）

### 3.1 价格结构

| 阶段 | 数量 | 单店年付 | 折扣 |
|---|---|---|---|
| **波 1 早鸟** | 前 10 家激活 | **¥199.9/校区/年** | 1 折 |
| **波 2 早鸟** | 11-30 家激活 | **¥999.5/校区/年** | 5 折 |
| **波 3 + 第 2 年起所有人** | 31+ 家 | **¥1,999/校区/年** | 正价 |

多校区机构：`¥1,999 × 校区数`（线性，无 SKU 阶梯）

### 3.2 早鸟规则

- 「前 X 家」= **激活的家数**（不是注册的）
- 早鸟客户**断订**后 slot 释放，新客户按当前波次补位
- 客户**不能跨波等待**（错过 1 折就只有 5 折）

### 3.3 全功能开放

无 SKU 阶梯。所有客户付 ¥1,999/校区/年 即可享受全部功能：
- 学员档案 / 老师业务卡 / 排课消课 / 课后点评 / 销售漏斗
- 财务（开票 / 退费 / 工资单） / 多校区报表 / API 接入
- 大区经理 / 跨校区财务清算 / 自定义品牌

## 四、C 端家长机制

### 4.1 Slot 池（免费体验）

- **10 个 slot/校区**（fixed pool，不刷新累积）
- **FCFS 先扫先得**
- 占用 slot = **3 个月免费**（从家长扫码起计时）
- 3 月到期 → 转付费 → slot 自动释放给下个家长

### 4.2 付费选项

| 选项 | 价格 | 周期 | 谁付 |
|---|---|---|---|
| **自付** | ¥9.9/月 | 按月，可随时退订 | 家长 |
| **集采** | ¥23.76/家长/季度（8 折）| 3 月起步，不能 1 年 | 机构 |

### 4.3 集采规则

- 仅机构付费才能享受 8 折
- 最低承诺 3 个月（季度计费，符合教培学期节奏）
- 不允许 1 年付集采（强制季度节奏）
- 不允许家长自付享受 8 折

## 五、推荐机制（V20 新增）★

### 5.1 设计哲学

**推荐 ≠ 点赞，推荐 = 朋友真来试听了。**

是产品最强差异化卖点：
- 别家 SaaS 没有
- 不能刷数据（必须 B 真试听 + 真评价才 +1）
- 业务卡显示"被 18 位家长真心推荐" = **销售杀手锏**

### 5.2 触发链路

```
家长 A（已是老师 X 的学员家长，已签约）
  ↓ 在 C 端老师 X 业务卡点「推荐给朋友」
  ↓ 生成专属小程序码（带 ref=A）
  ↓ 私聊发给真朋友 B（不公开朋友圈）
家长 B 微信扫码进入产品
  ↓ 申请试听老师 X
  ↓ 完成试听课
  ↓ 在 c/rate-teacher 给 5 表情评价
✅ 触发：老师 X 业务卡 referredCount +1
✅ V20 表记录：A 成功推荐 B 试听 X
```

### 5.3 显示位置

```
老师业务卡 profile 顶部 stats（与评分并排）:

  ⭐ 4.9 (12 评价)
  ❤️ 18 位家长真心推荐    ← V20 新增

老师业务卡推荐墙（V17 现有）:
  保留所有"家长推荐文字 + 评分"，并存展示
```

### 5.4 风控规则

| 维度 | 规则 |
|---|---|
| A 资格 | 必须是该老师的现有学员家长（FK 校验） |
| B 唯一 | 同一 B 只算 1 次（UNIQUE 约束）|
| A 次数 | 不限（A 可推荐多个不同的 B） |
| 防刷 | UNIQUE(referrer_parent_id, referee_parent_id, teacher_id) |

### 5.5 奖励机制

**先无奖励，纯口碑** —  上线后跑 1-2 个月，验证机制有效再加奖励（如送课时券）。
